# Ajan Koordinasyon Kuralları

1. **Dikey Delegasyon**: Liderlik ajanları departman liderlerine, departman liderleri uzmanlara devrededer. Karmaşık kararlar için bir katmanı asla atlama.
2. **Yatay Danışma**: Aynı kademedeki ajanlar birbirlerine danışabilir ancak kendi alanları dışında bağlayıcı kararlar alamaz.
3. **Çatışma Çözümü**: İki ajan anlaşmazlığa düştüğünde, ortak üste iletin. Ortak üst yoksa, tasarım çatışmaları için `creative-director`'a, teknik çatışmalar için `technical-director`'a iletin.
4. **Değişiklik Yayılımı**: Bir tasarım değişikliği birden fazla alanı etkilediğinde, `producer` ajanı yayılımı koordine eder.
5. **Tek Taraflı Alan Dışı Değişiklik Yasağı**: Bir ajan, açık delegasyon olmaksızın belirlenmiş dizinleri dışındaki dosyaları asla değiştiremez.
