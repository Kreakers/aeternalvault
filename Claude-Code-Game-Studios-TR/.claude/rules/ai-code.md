# Yapay Zeka Kod Kuralları

- Yapay zeka güncelleme bütçesi: kare başına maksimum 2ms — profilleme ile doğrulayın
- Tüm yapay zeka parametreleri veri dosyalarından ayarlanabilir olmalıdır (davranış ağacı ağırlıkları, algı mesafeleri, zamanlayıcılar)
- Yapay zeka hata ayıklanabilir olmalıdır: tüm yapay zeka durumu için görselleştirme kancaları uygulayın (yollar, algı konileri, karar ağaçları)
- Yapay zeka niyetlerini önceden belli etmelidir — oyuncuların okuyup tepki verebilmesi için zamana ihtiyaçları vardır
- Sabit kodlanmış if/else zincirlerine karşın fayda tabanlı veya davranış ağacı yaklaşımlarını tercih edin
- Grup yapay zekası; veri üzerinden formasyon, kuşatma ve rol atamasını desteklemelidir
- Tüm yapay zeka durum makineleri, hata ayıklama amacıyla geçişleri kaydetmelidir
- Ağdan gelen yapay zeka girdisine doğrulama yapmadan asla güvenmeyin
